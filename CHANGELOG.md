# 更新日志

本项目遵循 [语义化版本](https://semver.org/)。

## [1.0.0] - 2024-01-01

### 新增功能
- 初始版本发布
- 实现实时服务器监控功能
- 支持多服务器管理
- 添加数据可视化图表
- 实现响应式界面设计
- 支持 Docker 部署

### 技术特性
- 后端：FastAPI + psutil
- 前端：原生 HTML5 + Chart.js
- 架构：前后端分离设计

## 版本规则

- **主版本号**：不兼容的 API 修改
- **次版本号**：向下兼容的功能性新增  
- **修订号**：向下兼容的问题修正
